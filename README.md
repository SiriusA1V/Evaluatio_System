# 評価システム

説明　：　社員評価が可能なウェブアプリケーション

環境　：　React, Material, MSSQL, Ubuntu, Apache2, Node.js

PATH
- FrontEnd  : ./src
- BackEnd : ./server


※セキュリティに関わる情報は削除されている

--------------------

## 大きな機能

- 管理者
  - ユーザー管理
  - 評価管理
  - 給与管理
- 評価者
  - 部署別の評価リスト確認
  - ステータスによる評価
- 被評価者
  - 自分の評価リスト確認
  - ステータスによる評価作成
- その他
  - 更新によるメール飛ばし
  - 点数の自動更新
